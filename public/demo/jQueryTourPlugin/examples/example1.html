Example 1

This is a simple example of the jTour plugin with one step

Always wrap your Tour with the $(document).ready(function(){ [YOUR TOUR] }); to execute it when the DOM is fully loaded!

Code

$(document).ready(function() {

	var tourdata = [];
	
	var mystep = {
		html: "Hello World"
	};
	
	tourdata.push(mystep);	
	
	var myTour = jTour(tourdata);
	myTour.start();

});

Shorter version

$(document).ready(function() {

	var myTour = jTour([{
		html: "Hello World"
	}]);
	myTour.start();

});
